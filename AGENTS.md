# Agentes de trabalho usados no projeto

## 1. PixelArtAgent
Responsável por:
- identidade visual dos blocos
- ícones de itens
- texturas do balde e armadura
- consistência cromática entre Escura, Clara e Amarela

## 2. ModProgrammingAgent
Responsável por:
- registries
- fluidos
- capability de infecção
- HUD
- overlay
- backend HTTP
- sincronização de rede
- spawn de focos de Matéria Escura

## 3. ModpackIntegrationAgent
Responsável por:
- recipes iniciais
- equilíbrio de progressão
- organização de creative tab
- documentação de instalação/configuração
- separação entre mecânica base e futura expansão
